# Inventory & Analytics System (Cloud Native)
Proyecto Full Stack desarrollado para demostrar competencias en arquitectura de microservicios y despliegue en la nube.

## Stack Tecnológico
* **Backend:** .NET 8 (C#) con Entity Framework Core.
* **Frontend:** React + TypeScript.
* **Microservicios:** TypeScript en AWS Lambda.
* **Base de Datos:** SQL Server (LocalDB / RDS).
* **Patrones de Diseño:** Outbox Pattern, Repository Pattern.
